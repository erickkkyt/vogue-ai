## 一句话判断

VogueAI 的 X 账号不要只学 MeiGen 品牌号，而要学两类账号的组合：高频图片 prompt 账号负责日更供给，workflow/video 账号负责爆点和收藏。MeiGen 自身更像产品中枢，真正的内容节奏要从 prompt creator 和 AI visual workflow creator 里学。

```text
日更 prompt 图文
-> 可替换变量 / prompt schema
-> 自生成图片或视频结果
-> source credit / creator board
-> 回复或置顶补 VogueAI prompt page 链接
```

## 筛选标准

只保留最值得对标的账号，筛选标准如下：

- 最近 24-72 小时内仍在更新。
- 主要内容有 AI 图片或 AI 视频结果，不只是文字观点。
- 帖子里能看到 prompt、workflow、变量、模型、工具或生成步骤。
- 对 VogueAI 的 prompt asset、X 日更、creator board、社媒回流有直接复用价值。
- 纯推广号、泛 AI 新闻号、开源热点号、大而泛的 AI 趋势号不放入主清单。

## 参考模板

### 模板 1：日更图片 prompt 帖

适合学习账号：`@iamaiistudio`、`@Naiknelofar788`、`@Taaruk_`、`@SimplyAnnisa`、`@liyue_ai`。

```text
GPT Image 2 on ChatGPT

[一句强视觉结果描述]

Prompt:
[完整 prompt 或核心 schema]

Variables:
[SUBJECT] =
[SCENE] =
[STYLE] =
[COMPOSITION] =
[CONSTRAINTS] =
```

执行要求：

- 主帖放 2-4 张自生成图。
- 主帖不要只放链接；链接放第一条回复或置顶帖。
- 如果 prompt 很长，主帖放核心 schema，回复放 full prompt。
- 每个模板至少做 2 条：结果帖和变量帖。

### 模板 2：同一 schema 多变体图组

适合学习账号：`@MrLarus`、`@Naiknelofar788`、`@liyue_ai`。

```text
Same visual recipe, 4 outputs.

Only the variables changed:

1. [CITY / SUBJECT / PRODUCT A]
2. [CITY / SUBJECT / PRODUCT B]
3. [CITY / SUBJECT / PRODUCT C]
4. [CITY / SUBJECT / PRODUCT D]

The reusable part is the schema, not one image.
```

执行要求：

- 一组图优先用 4 张，便于 X 网格展示。
- 配图要来自 VogueAI / 自有生成结果。
- 变量必须可读，方便用户收藏和复用。
- 对应 VogueAI 模板库要回写 `sourceHandle`、`sourceUrl`、`schemaVariables`。

### 模板 3：workflow / video thread

适合学习账号：`@aimikoda`、`@underwoodxie96`、`@songguoxiansen`、`@Dheepanratnam`、`@ivanka_humeniuk`。

```text
I turned one image idea into a short AI video workflow.

Stack:
1. Base image: [model/tool]
2. Character/reference: [method]
3. Motion: [video model]
4. Edit/polish: [tool]

Workflow and prompt in replies.
```

执行要求：

- 主帖先给视频或最强结果，不先解释产品。
- 回复 1 放生成步骤。
- 回复 2 放 prompt/schema。
- 回复 3 再放 VogueAI prompt page 或 gallery 链接。
- 每周 1-2 条即可，不要每天硬做。

### 模板 4：creator board / source credit 帖

适合学习账号：`@meigen7982`、`@underwoodxie96`、`@AmirMushich`、`@songguoxiansen`。

```text
Creator board: @[source_handle]

I turned this visual idea into a reusable prompt schema:

- intent:
- subject:
- scene:
- style:
- constraints:

Original source credited below.
```

执行要求：

- 只做 inspired-by，不搬运原图当自有结果。
- 保留 source URL、source handle、collected date。
- 用 VogueAI 自生成图做发布媒体。
- 如果要 tag 创作者，优先用 quote/reply 方式，避免像硬蹭。

### 模板 5：产品中枢帖

适合学习账号：`@meigen7982`。

```text
New in VogueAI:

You can now turn a prompt idea into:

1. image result
2. reusable schema
3. source credit
4. prompt page
5. generation workflow

The prompt gallery is becoming a visual recipe library.
```

执行要求：

- 每周最多 1 条产品中枢帖。
- 不要把日更都发成产品公告。
- 这种帖用于解释 VogueAI 的长期定位，不用于冷启动拿自然流量。

## 不进入主对标清单的边界

- 纯推广/付费合作号：可以研究投放话术，不学日常内容。
- 泛 AI 新闻/开源热点号：可以研究扩散渠道，不学 prompt 日更。
- 大而泛的 AI 趋势号：可以看爆点，不作为冷启动账号模板。
- 纯产品公告号：可以学机制，不学发帖节奏。

## 执行边界

1. 不直接搬运竞对 prompt 和原图；只做 inspired-by、变量化、结构化和自生成媒体。
2. 主帖优先站内原生价值，链接放回复、置顶或 thread 末尾。
3. 每个 X 内容都应该回到 VogueAI 的资产字段：`sourceHandle`、`sourceUrl`、`promptSchema`、`generatedMedia`、`promptPageUrl`、`xPostUrl`。
4. 判断账号是否值得继续跟踪，看 bookmark、reply、profile visit 和是否带来 prompt asset，而不是只看 views。
