## 一句话判断

Vogue AI 不应该只做一个类似 MeiGen 的品牌号。更好的结构是：

```text
品牌号 @VogueAI
-> 承接产品可信度、prompt gallery、creator credit、转化链接

AIGC 创作者号
-> 用人格化账号持续发 AI image / AI video / prompt remix / model comparison
-> 用自然内容拿曝光，再把流量导回 Vogue AI
```

原因很直接：品牌号很难天然起量，尤其是新账号；AIGC 创作者号更容易通过视觉 demo、prompt thread、before/after、模型对比拿自然传播。MeiGen 的增长也不是靠 `@meigen7982` 自己发爆，而是靠外部 AI creator、prompt 内容源、署名机制和推广号扩散。

## 账号矩阵

品牌号负责“可信”和“转化”，不是承担全部自然流量。

建议频率：

| 动作 | 频率 | 目的 |

| 原帖 | 每天 1-2 条 | 发布精选 prompt、产品更新、creator board、模型趋势 |
| 转发 / quote | 每天 3-5 条 | 维护创作者关系，放大使用案例 |
| 回复 | 每天 10-20 条 | 在 AI image / AI video 讨论区露出 |
| 长 thread | 每周 2-3 条 | 做 prompt pack、模型对比、工作流总结 |

品牌号内容比例：

```text
40% 精选 prompt gallery / creator board

15% 用户案例、教程、转发和合作
```

### 2. AIGC 创作者号

这个号可以透明写 `Building Vogue AI`，但日常内容不要像广告号。它应该像一个 AI visual creator，而不是公司公告栏。

建议频率：

| 动作 | 频率 | 目的 |
| --- | ---: | --- |
| 单图 / 短视频 demo | 每天 2-3 条 | 拿视觉停留和转发 |
| prompt thread | 每天 1 条 | 做收藏和关注 |
| model comparison | 每周 3-4 条 | 借模型热点提高传播 |
| 高质量回复 | 每天 20-30 条 | 去大号评论区截流 |
| quote creator | 每天 3-5 条 | 建立创作者关系 |

创作者号内容比例：

```text
50% 原创/改造后的 AI 视觉结果
20% prompt 拆解和可复制模板
15% 模型对比、工具工作流
10% trend remix / creator remix
5% Vogue AI 产品入口
```

关键原则：这个号必须先让人觉得“关注它能学到 AI 视觉玩法”，然后再顺手把 Vogue AI 作为生成和整理工具带出来。

## 这些账号主要发什么

高互动 AIGC 内容的形态非常固定，Vogue AI 可以直接按这些格式自动化生产。

| 内容类型 | 标题/开头方式 | 素材要求 | 适合平台 |
| --- | --- | --- | --- |
| Viral demo | `This AI image prompt is wild...` | 1 张强结果图或 5-8 秒视频 | X / TikTok / YouTube Shorts |
| Prompt thread | `10 prompts for cinematic product photos` | 8-12 个 prompt + 结果图 | X / Pinterest |
| Before / after | `I turned this product photo into a luxury campaign` | 原图、结果图、prompt | X / TikTok / Pinterest |
| Model comparison | `Same prompt in GPT Image vs Midjourney vs Nano Banana` | 同 prompt 多模型结果 | X / YouTube Shorts |
| Workflow breakdown | `Claude + Vogue AI made this in 3 minutes` | 过程截图、prompt、结果 | X / YouTube |
| Trend remix | `I remixed this viral prompt into a fashion campaign` | 来源 credit + VogueAI 二创结果 | X / TikTok |
| Soft sponsored post | `I found a free gallery for viral AI image prompts` | 产品截图 + 结果图 + CTA | X |

不要把每天内容写成“我们上线了什么”。用户更关心“我能不能复制这个效果”。

## 原创与搬运边界

Vogue AI 可以围绕 X 上的 prompt 做内容，但不要直接把别人的原图、原文案、原 prompt 当自己的原创发。

建议标准：

1. 保留来源字段：`sourceUrl`、`sourceAuthor`、`sourceHandle`、`sourceCollectedAt`。
2. 社媒发布时写 `inspired by @xxx` 或 `source: @xxx`，尤其是明显来自某个创作者的 prompt。
3. 用 Vogue AI 重新生成结果图，并做实质变化：角色、场景、构图、色彩、行业、镜头语言至少改 2-3 个维度。
4. 不发布“100% 复刻某创作者作品”的说法，改成 `remix`、`variation`、`style experiment`。
5. 保留 takedown 入口：如果创作者不希望被收录，可以联系删除。

推荐生产链路：

```text
X / Reddit / Pinterest 发现爆款视觉 prompt
-> 记录来源和互动数据
-> 在 Vogue AI 里生成 3-5 个变体
-> 选最强结果图
-> 生成 X thread / TikTok script / Pinterest pin
-> 发布时保留 credit 和 Vogue AI 链接
```

## 第一轮投放建议

第一轮不要买太多大号。目标不是“曝光最大化”，而是找到能转化的素材、话术和账号类型。

建议预算：`$5,000-$8,000`

配置：

| 类型 | 数量 | 单价 | 目的 |
| --- | ---: | ---: | --- |
| 小号 / 垂直号 | 5 个 | `$100-$300` | 测试话术、地区、素材 |
| 中腰部 AI creator | 5 个 | `$400-$800` | 测试真实点击和注册 |
| 稍大号 | 1 个 | `$1,500-$3,000` | 测试上限和品牌势能 |
| creator board 合作 | 1-2 个 | `$500-$2,000` | 换取可长期复用的内容资产 |

付款结构优先：

```text
固定费用 + affiliate
例：$300-$800 base + 20%-30% 首月收入分成
```

不要只买“发一条”。最好买：

```text
1 条主推文
+ 1 条 thread 或 follow-up
+ 品牌号可 quote/repost
+ 允许我们把内容截图/链接放到网站或后续社媒
+ 发后 24h 和 72h 数据截图
```

## 投放脚本要求

每个合作账号都必须提前确认 brief。

必填项：

```text
1. 目标链接：带 UTM 的 Vogue AI 落地页
2. 核心话术：不要超过 1 个主卖点
3. 素材：至少 1 张由 Vogue AI 生成的强结果图
4. CTA：copy prompt / browse gallery / generate your own version 三选一
5. 披露：按当地规则标注 sponsor / ad / partner
6. 数据回传：24h、72h views / likes / replies / reposts / bookmarks / link clicks
```

推荐话术：

```text
I found a Pinterest-style gallery for AI image prompts.

You can browse real examples, copy the prompt, and generate your own version.
No prompt engineering rabbit hole.

Built for people who save AI image posts and never find them again.
```

更适合 Vogue AI 的变体：

```text
Stop saving AI image prompts in random bookmarks.

Vogue AI turns real prompt examples into a searchable gallery:
- see the final image
- copy the prompt
- remix the style
- generate your own version
```

## 自动化生产链路

可以用 AI 自动化批量产出，但人要把最后的审美关住。

推荐链路：

```text
输入层：
X bookmarks / X search / Reddit / Pinterest / TikTok trend

结构化层：
sourceUrl
sourceAuthor
sourceHandle
sourcePlatform
sourceViews
sourceLikes
sourceBookmarks
promptText
visualCategory
modelTag

生成层：
Vogue AI 生成 3-5 个变体
保留原 prompt 的灵感，但替换角色、场景、行业和构图

包装层：
生成 X 单帖
生成 X thread
生成 TikTok / Shorts script
生成 Pinterest title + description

分发层：
Buffer / Typefully / Hypefury / Metricool 等排程
所有链接加 UTM

复盘层：
记录 postId、views、bookmarks、clicks、signups、paid conversions
```

最重要的不是“每天发很多”，而是建立一个内容资产库。每条社媒内容都应该回流成：

```text
prompt gallery card
creator board
SEO prompt detail page
Pinterest pin
short video asset
future ad creative
```

## 判断账号是否值得买

优先级排序：

1. 最近 10 条平均 views 高于粉丝数的 10%-20%。
2. bookmark 比例高，说明用户真的想保存 prompt。
3. 账号平时就发 AI image / AI video / prompt / tool，不是泛科技号。
4. sponsor 内容没有明显掉互动。
5. 能提供历史 sponsor 案例和后台截图。
6. 愿意自己试用 Vogue AI 生成结果，而不是只复制广告文案。

不建议买：

1. 粉丝大但每条只有几百 views。
2. 每天全是广告转发。
3. 没有 AI 视觉受众，只是泛商业/泛科技。
4. 不愿意给数据截图。
5. 只接受硬广文案，不愿意做 demo。

## 当前最可执行方案

1. 立刻开一个 AIGC 创作者号，透明绑定 Vogue AI，但内容先做视觉 creator，不做公告号。
2. 品牌号每天稳定发 1-2 条，创作者号每天发 3-5 条。
3. 先用 30 天跑内容，找到 5 个高 bookmark / 高点击的内容模板。
4. 第一轮付费只投 `$5,000-$8,000`，主测中腰部，不急着买 Hasan 这种大号。
5. 每个推广账号都要求 Vogue AI 生成图、UTM、24h/72h 数据截图。
6. 把表现好的内容回流到站内：prompt detail、creator board、Pinterest、短视频、SEO 页面。

最终目标不是“社媒账号看起来热闹”，而是形成这个闭环：

```text
社媒发现爆款视觉玩法
-> Vogue AI 生成 remix
-> 创作者号发布并拿自然曝光
-> 品牌号承接转化
-> 站内沉淀 prompt gallery / creator board
-> 付费推广号放大已验证素材
-> 数据反哺下一轮选题
```
